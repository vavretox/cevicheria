5. Tecnologías Utilizadas

Stack tecnológico completo del proyecto

Backend - Laravel 11
  • MVC Pattern
  • Eloquent ORM
  • Migrations & Seeders
  • Middleware de roles
  • Blade Templates
Base de Datos - MySQL
  • Tablas: users, categories, products, orders, order_details
  • Relaciones bien definidas
  • Índices optimizados
  • Datos de prueba incluidos
Frontend - Bootstrap 5
  • Diseño responsivo
  • Componentes modernos
  • Grid system
  • Utilities classes
JavaScript - jQuery + AJAX
  • Interactividad dinámica
  • Llamadas AJAX
  • Validaciones en tiempo real
  • Carrito de pedidos interactivo
Características Adicionales:
  • Sistema de autenticación completo
  • Middleware de roles (Admin, Cajero, Mesero)
  • Carga de imágenes de productos
  • Generación de boletas imprimibles
  • Cálculo automático de IGV (18%)
  • Reportes de ventas con filtros
  • Seeders con datos de prueba
Instalación:
composer install
npm install && npm run build
php artisan migrate --seed
php artisan storage:link
php artisan serve
